Different Databases = Different Information

Different databases search different sources and types of information. Match the type of information you need to the database you choose.

Understanding the Differences

Be sure you know the difference between the database name and the database provider that hosts the database. For example, EBSCOhost is the name of the database provider that hosts Academic Search Complete but it also provides access to many other databases.
